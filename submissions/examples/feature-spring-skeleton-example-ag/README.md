# Spring Skeleton Example

A standalone CSS skeleton loader example with a smooth spring animation effect.

## Features

- Spring-based skeleton animation
- Disabled/loading state UI pattern
- Accessible loading indication using ARIA attributes
- Supports `prefers-reduced-motion`
- Fully isolated inside submissions folder
- Uses required `-ag` suffix naming

## Usage

Add the HTML structure and include the CSS file.

```html
<div class="spring-skeleton-example-ag">Loading content...</div>
```
