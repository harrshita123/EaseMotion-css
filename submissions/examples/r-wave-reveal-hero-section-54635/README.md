# CSS Wave-Reveal Hero Section

A modern **Wave-Reveal Hero Section** built using pure HTML5 and CSS3 for creative portfolio layouts. This example features a smooth wave-style reveal animation, floating layered panels, responsive design, and accessibility support.

---

## ✨ Features

- Pure HTML5 & CSS3
- Wave-reveal entrance animation
- Floating layered visual panels
- Interactive hover effects
- Responsive two-column hero layout
- CSS Custom Properties
- Modern gradient buttons
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the component into your project.
2. Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

3. Open `demo.html` in any modern browser.
