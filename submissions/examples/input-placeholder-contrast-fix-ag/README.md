# Placeholder Contrast Fix (Issue #14080)\n\n## What does this do?\nRemoves `opacity: 0.7` from the placeholder style and uses a slightly darker muted color (`#6b7280`) at full opacity to maintain WCAG 1.4.3 minimum contrast.\n\n## How is it used?\n```html\n<input class="ease-input" placeholder="Enter your name">\n```\n\n## Why is it useful?\n`opacity: 0.7` multiplied against `#64748b` drops the effective contrast ratio below 3:1, failing WCAG. Using `opacity: 1` with a darker base color (`#6b7280`) achieves the same visual style while meeting accessibility contrast requirements.
