# Glassmorphic Pricing Toggle Card

A responsive, high-contrast glassmorphic pricing card deck built for EaseMotion CSS featuring an interactive monthly/yearly billing toggle switch, elevation micro-interactions, and visual emphasis ribbons.

## 1. What does this do?
This component renders a sleek glassmorphic pricing comparison table with backdrop blur filters and subtle neon gradient highlights. Toggling the monthly/yearly switch dynamically updates tier rates with smooth numeric transition feedback.

## 2. How is it used?
1. Link `style.css` in your HTML template `<head>`.
2. Wrap your plans inside `.pricing-grid` and utilize `.pricing-card` elements.
3. Bind a click or change handler to `#billingToggle` to swap `data-monthly` and `data-yearly` values dynamically.

## 3. Why is it useful?
- **Modern SaaS Aesthetics**: Enhances conversion rates using glassmorphism, gradient accents, and hover elevation physics.
- **Responsive Layout**: Flexes seamlessly across mobile screens, tablets, and desktop displays.
- **Pure CSS Base**: Lightweight, performant styling with zero external dependencies.
