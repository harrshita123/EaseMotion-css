# CSS Bounce-Pulse Hero Section

A modern **Bounce-Pulse Hero Section** built with pure HTML5 and CSS3 for creative portfolio layouts. This example combines a bounce entrance animation with pulsing decorative elements to create an engaging landing section while remaining lightweight, responsive, and accessible.

---

## ✨ Features

- Pure HTML5 & CSS3
- Bounce entrance animation
- Pulse background animation
- Interactive hover effect
- Responsive hero layout
- CSS Custom Properties
- Modern gradient buttons
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the component into your project.
2. Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

3. Open `demo.html` in your browser.
