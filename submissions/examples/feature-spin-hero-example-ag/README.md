# Spin Hero Example

## Description
This example demonstrates a simple hero element that performs a smooth 360-degree spin animation on hover.

## Features
- Smooth hover animation
- Beginner-friendly example
- Uses CSS transitions
- Supports reduced motion preferences

## Usage
Open `demo.html` in your browser and hover over the hero element to see the spin effect.
